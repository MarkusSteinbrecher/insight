# Citation Format Templates

## Full Source Attribution

```
Author Last, First. "Title of Work." Publication/Publisher, Date. URL
```

### Examples by Source Type

**Article/Blog:**
```
Smith, Jane. "The Rise of AI Agents in Enterprise." InfoQ, March 15, 2024. https://example.com/article
```

**Report:**
```
Gartner. "Hype Cycle for Artificial Intelligence, 2024." Gartner Research, July 2024. https://example.com/report
```

**Academic Paper:**
```
Chen, L., Wang, M., & Park, S. "Evaluating Autonomous Agent Reliability." Proceedings of ACM CHI 2024. https://doi.org/example
```

**Video/Podcast:**
```
Ng, Andrew. "Building AI Agents." [Video]. DeepLearning.AI, April 2024. https://example.com/video
```

**Documentation:**
```
Anthropic. "Claude API Documentation — Tool Use." Accessed January 2025. https://docs.anthropic.com/tool-use
```

## Inline Citation Formats

### Blog Post Style
```markdown
[descriptive text](URL)
```

### Endnote Style
```markdown
Text making a claim [1].

## References
[1] Full attribution here.
```

### Footnote Style
```markdown
Text making a claim.¹

---
¹ Abbreviated attribution.
```

## Bibliography Section Template

```markdown
## References

[1] Last, First. "Title." Publisher, Date. URL
[2] Last, First. "Title." Publisher, Date. URL
[3] Organization. "Title." Date. URL
```

## Multiple Source Citation

When a claim is supported by multiple sources:
```markdown
Multiple analysts project significant growth in AI agent adoption [1, 3, 7].
```

## Data Citation

When citing specific data points:
```markdown
Enterprise AI agent adoption reached 60% in 2024 [1, p. 12], up from 35% the prior year [1, p. 8].
```
